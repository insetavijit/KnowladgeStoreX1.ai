# 3.1.4 Collection Items

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Understanding the structure of the objects returned in a collection array, enabling access to page metadata, content, and file paths within loops. |
| **Key Concepts / Details** | - **Data Object**: `item.data` contains front matter and global data cascade values.<br>- **Page Object**: `item.page` contains `url`, `date`, `inputPath`, `outputPath`.<br>- **Template Content**: `item.templateContent` holds the rendered HTML string.<br>- **Raw Input**: `item.inputPath` for file system reference.<br>- **Previous/Next**: Linking to `item.url` for navigation. |
| **One-Line Recall** | The specific data structure (url, date, data, content) of each item in a collection array. |
