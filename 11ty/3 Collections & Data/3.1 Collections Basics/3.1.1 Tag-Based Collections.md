# 3.1.1 Tag-Based Collections

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Understanding how to group content automatically using the simple `tags` front matter key, enabling easy iteration and display of related content groups. |
| **Key Concepts / Details** | - **Defining Tags**: Adding `tags: "blog"` or `tags: ["post", "tech"]` to front matter.<br>- **The `collections` Object**: Accessing tagged items via `collections.tagName`.<br>- **Iteration**: Looping through `collections.tagName` in templates to display lists.<br>- **String vs Array**: Tags can be a single string or an array of strings.<br>- **Case Sensitivity**: Tags are case-sensitive strings. |
| **One-Line Recall** | The simplest way to group content by adding a `tags` key to front matter. |
