# 3.1.3 Custom API Collections

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Using `eleventy.config.js` to create advanced collections programmatically, allowing for arbitrary filtering, grouping, and sorting logic beyond simple tags. |
| **Key Concepts / Details** | - **`addCollection` API**: `eleventyConfig.addCollection("name", collection => ...)`<br>- **Filtering**: Returning a strict subset of `collection.getAll()`.<br>- **Data Access**: Logic can access front matter data, file paths, or external data.<br>- **Flexibility**: Create collections based on file extension, folder, or complex metadata criteria.<br>- **Memoization**: Collections are computed once per build. |
| **One-Line Recall** | Programmatically defining collections in the config file for complex grouping logic. |
