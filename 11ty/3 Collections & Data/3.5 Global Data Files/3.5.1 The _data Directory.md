# 3.5.1 The `_data` Directory

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Understanding the specific role of the `_data` folder in the project root as the source of global data accessible to all templates. |
| **Key Concepts / Details** | - **Location**: `src/_data` (or strict root `_data` if `input` dir is root).<br>- **Behavior**: Files in this directory automatically become global data objects.<br>- **Naming**: `_data/site.json` becomes `site` variable.<br>- **Subdirectories**: `_data/users/list.json` becomes `users.list`.<br>- **Config**: Changing the location via `dir.data` in eleventy config. |
| **One-Line Recall** | The special directory where files serve as globally accessible data objects. |
