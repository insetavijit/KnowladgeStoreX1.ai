# 3.5.4 Fetching External Data

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Pulling data from remote APIs (Headless CMS, Weather API, GitHub, etc.) during the build process to populate the site. |
| **Key Concepts / Details** | - **Async Exports**: `module.exports = async function() { ... }`.<br>- **Fetch API**: Using native `fetch` or `axios` inside the JS data file.<br>- **Await**: Pauses the build until data is received.<br>- **Error Handling**: `try/catch` blocks to prevent build failure on API glitch.<br>- **Integration**: The bridge between static Eleventy and dynamic content sources. |
| **One-Line Recall** | Using async JavaScript data files to request and consume remote API data. |
