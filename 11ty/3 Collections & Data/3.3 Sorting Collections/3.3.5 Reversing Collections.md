# 3.3.5 Reversing Collections

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Inverting the order of a collection, most commonly used to display the newest blog posts first (descending date order). |
| **Key Concepts / Details** | - **Filter Usage**: `{% for item in collections.posts | reverse %}`.<br>- **API Usage**: `.reverse()` array method in `addCollection` (mutates array!).<br>- **Mutation Warning**: Be careful mutating the source array in JS; copy it first (`[...arr].reverse()`).<br>- **Pagination**: Reversing in front matter `pagination` settings (`reverse: true`).<br>- **Visuals**: often paired with `limit` to show "Recent Posts". |
| **One-Line Recall** | Flipping the collection order, typically to show newest items first. |
