# 3.8.5 Remapping Permalinks

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Customizing the URL structure of paginated pages, especially when aliasing, to create semantic URLs (e.g., `/pokemon/pikachu/`) instead of numbered pages. |
| **Key Concepts / Details** | - **Dynamic Syntax**: `permalink: "pokemon/{{ pokemon.name }}/"` (if interacting with alias).<br>- **Slugify**: `{{ pokemon.name | slug }}` ensures URL safety.<br>- **Page Numbering**: `permalink: "blog/page-{{ pagination.pageNumber | plus: 1 }}/"`.<br>- **Handling Index**: `{% if pagination.pageNumber == 0 %}index.html{% else %}...{% endif %}`.<br>- **Consistency**: Ensuring unique permalinks for every generated page. |
| **One-Line Recall** | Rewriting the output URL for paginated/aliased pages using data variables. |
