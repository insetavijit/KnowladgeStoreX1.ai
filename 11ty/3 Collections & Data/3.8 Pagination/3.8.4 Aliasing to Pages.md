# 3.8.4 Aliasing to Pages

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Generating a single individual page for *every item* in a dataset (e.g., creating a dedicated page for every Pokemon in a JSON file). |
| **Key Concepts / Details** | - **Size 1**: `size: 1` means one item per page.<br>- **Alias**: `alias: pokemon` creates a variable `pokemon` representing that single item.<br>- **Template**: The template is rendered once for each item.<br>- **Use Case**: transforming a flat data file (tags.json, catalog.json) into distinct URLs.<br>- **Flattening**: No need to loop `pagination.items` since it's just one item. |
| **One-Line Recall** | Setting `size: 1` and `alias` to generate individual detail pages from a data array. |
