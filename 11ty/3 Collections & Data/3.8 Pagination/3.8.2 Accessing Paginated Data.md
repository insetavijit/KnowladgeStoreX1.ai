# 3.8.2 Accessing Paginated Data

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Retrieving the specific subset of items assigned to the current page within a paginated template. |
| **Key Concepts / Details** | - **Current Chunk**: `pagination.items` is the array of items for *this* page.<br>- **Iteration**: `{% for post in pagination.items %}`.<br>- **Page Info**: `pagination.pageNumber` (0-indexed).<br>- **Total**: `pagination.hrefs.length` gives total page count.<br>- **Context**: The rest of the page data (title, layout) remains constant across pages. |
| **One-Line Recall** | Using `pagination.items` to access the specific slice of data allocated to the current page. |
