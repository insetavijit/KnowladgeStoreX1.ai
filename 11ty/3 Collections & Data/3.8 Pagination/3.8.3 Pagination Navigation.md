# 3.8.3 Pagination Navigation

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Building "Previous" and "Next" links or a numbered page navigation list to allow users to browse through paginated content. |
| **Key Concepts / Details** | - **Next/Prev**: `pagination.href.next` and `pagination.href.previous`.<br>- **Links Array**: `pagination.hrefs` contains URLs for all pages.<br>- **Current**: Identifying strict equality with `page.url` to style "active" state.<br>- **First/Last**: `pagination.href.first` / `last`.<br>- **Conditionals**: `{% if pagination.href.next %}` to hide dead links. |
| **One-Line Recall** | Generating the UI controls for moving between paginated pages using `href` properties. |
