# 3.7.1 `eleventyComputed.js`

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Enabling "Late Binding" of data properties, defaulting to values calculated at the end of the data cascade, allowing front matter to depend on other data. |
| **Key Concepts / Details** | - **Activation**: Adding `eleventyComputed:` key to front matter or global data.<br>- **Syntax**: Defining keys inside this object that resolve later.<br>- **Usage**: `title: "{{ otherVar }} - My Title"`.<br>- **Direct Access**: `data.title` or `data.page.url` is available inside computed functions in JS.<br>- **Global File**: Using `eleventyComputed.js` in a directory to apply to all files. |
| **One-Line Recall** | The feature that allows data properties to wait until all other data is resolved before calculating their own values. |
