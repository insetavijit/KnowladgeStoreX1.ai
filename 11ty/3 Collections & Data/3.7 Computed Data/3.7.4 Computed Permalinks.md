# 3.7.4 Computed Permalinks

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Dynamically generating the output URL (`permalink`) for a page based on its content, metadata, or external data. |
| **Key Concepts / Details** | - **Dynamic Routing**: `permalink: "posts/{{ title | slug }}/index.html"`.<br>- **SEO**: Creating clean URLs from arbitrary data fields.<br>- **Redirects**: conditionally setting permalink to `false` to prevent output.<br>- **Pagination**: Often used with pagination to create `/page/1/`, `/page/2/`.<br>- **Date-Based**: `permalink: "/{{ date | date: '%Y/%m' }}/{{ slug }}/"`. |
| **One-Line Recall** | Using computed data to programmatically define the final output URL for a template. |
