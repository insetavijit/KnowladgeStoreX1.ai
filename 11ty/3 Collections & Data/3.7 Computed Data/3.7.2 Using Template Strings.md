# 3.7.2 Using Template Strings

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Leveraging supported template syntaxes (like Liquid or Nunjucks) directly within front matter values via `eleventyComputed`. |
| **Key Concepts / Details** | - **Interpolation**: `key: "Hello {{ name }}"` works if `name` is in the cascade.<br>- **Filters**: `title: "{{ rawTitle | safe }}"` can use filters.<br>- **Conditionals**: `active: "{% if page.url == '/' %}true{% endif %}"`.<br>- **Context**: The string rendering has access to the page's full data context.<br>- **Evaluation**: processed just before the actual template rendering. |
| **One-Line Recall** | Embedding template logic (variables, filters) inside front matter values using computed data. |
