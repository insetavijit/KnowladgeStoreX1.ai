# 3.2.1 Filtering with Array Methods

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Using standard JavaScript array methods like `.filter()` within `addCollection` to create precise subsets of content based on simple conditions. |
| **Key Concepts / Details** | - **Standard JS**: Utilizing `Array.prototype.filter()`.<br>- **Conditionals**: Returning `true` to keep an item, `false` to discard.<br>- **Data Access**: Filtering based on `item.data.prop`.<br>- **Chaining**: Combining `.filter().sort().map()`.<br>- **Example**: `collection.getAll().filter(item => item.data.published)`. |
| **One-Line Recall** | Applying standard JS `.filter()` logic to the collection array in the config file. |
