# 3.6.5 Troubleshooting Scope

| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | Diagnosing issues where directory data isn't applying as expected, typically due to naming mismatches or location errors. |
| **Key Concepts / Details** | - **Naming Match**: The data file MUST match the folder name exactly (unless using `11tydata.js`).<br>- **Location**: Must be *inside* the folder, not beside it (usually).<br>- **Extension**: Checking `.json` vs `.11tydata.js` vs `.11tydata.json`.<br>- **Syntax Errors**: Invalid JSON will fail silently or break the build.<br>- **Debug Filter**: Using `{{ myVar | log }}` in a template to see if the value arrived. |
| **One-Line Recall** | Common pitfalls and fix strategies for ensuring directory data is correctly applied. |
