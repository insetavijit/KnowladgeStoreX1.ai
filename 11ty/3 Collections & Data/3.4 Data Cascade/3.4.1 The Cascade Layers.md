| Section | Details |
| :--- | :--- |
| **Focus & Purpose** | visualizing the hierarchy of data sources in Eleventy, understanding which source overrides which when keys collide. |

| **One-Line Recall** | The specific priority order (Global < Directory < Layout < Front Matter < Computed) of data merging. |
