**Overview**  
Theme setup is the process of declaring what WordPress features your theme supports, typically done in a function hooked to `after_setup_theme`. This includes enabling post thumbnails, HTML5 markup, custom logos, title tag support, automatic feed links, and more. Using `add_theme_support()` tells WordPress which built-in features to activate for your theme, ensuring compatibility with core functionality and plugins. Proper theme setup is essential because it enables features users expect (like featured images), improves SEO (with proper title tags), and ensures your theme follows WordPress standards. Understanding theme supports is a fundamental requirement for building professional, feature-complete themes.
