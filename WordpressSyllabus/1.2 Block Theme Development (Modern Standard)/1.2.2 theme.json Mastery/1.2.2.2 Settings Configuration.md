| **Section**                            | **Interview Learning Goals**                                                   | **Key Technical Details to Study**                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **Settings Configuration**             | Explain how to control the editor experience via theme.json.                   | Enabling/disabling UI controls; managing color/typography/spacing options; understanding the `settings` key.|
| **[[Settings vs Styles Recap]]**       | Briefly reinforce the distinction.                                             | Settings = UI controls availability; Styles = Default values applied to blocks.                             |
| **[[Appearance Tools]]**               | Explain the `appearanceTools` shortcut.                                        | `settings.appearanceTools: true`; opt-in for border, color, spacing, typography, dimensions controls all at once.|
| **[[Color Settings]]**                 | Describe how to manage color controls.                                         | `settings.color.text`, `background`, `link`; `palette` (custom vs default); `gradients`; `duotone`.         |
| **[[Typography Settings]]**            | Explain typography control management.                                         | `settings.typography.fontSizes`, `fontFamilies`; `customFontSize`, `dropCap`, `fontStyle`, `fontWeight`.    |
| **[[Spacing Settings]]**               | Describe enabling spacing controls.                                            | `settings.spacing.padding`, `margin`, `blockGap`; `customSpacingSize`; implementation of spacing scale.     |
| **[[Layout Settings]]**                | Explain content width and layout controls.                                     | `settings.layout.contentSize`, `wideSize`; `allowEditing`; relationship to `align` specific blocks.         |
| **[[Border & Shadow]]**                | Describe border and shadow controls.                                           | `settings.border.color`, `radius`, `style`, `width`; `settings.shadow.presets`.                             |
| **[[Custom Settings]]**                | Show how to add arbitrary data for use in blocks/patterns.                     | `settings.custom.myKey`; accessing via `var(--wp--custom--my-key)`; use cases for unique theme data.        |
| **[[Disabling Core Defaults]]**        | Explain how to restrict the editor for stricter design systems.                | `settings.color.defaultPalette: false`; `settings.typography.dropCap: false`; locking down the UI.          |
| **[[Block-Specific Settings]]**        | specific granular control per block type.                                      | `settings.blocks.core/heading.typography.fontSize: true`; overriding global settings for specific blocks.   |
