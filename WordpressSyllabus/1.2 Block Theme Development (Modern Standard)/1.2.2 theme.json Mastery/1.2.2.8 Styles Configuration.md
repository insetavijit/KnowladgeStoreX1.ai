| **Section**                            | **Interview Learning Goals**                                                   | **Key Technical Details to Study**                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **Styles Configuration**               | Explain how to define default block styling via theme.json.                    | `styles` key; elements, blocks, typography, color; CSS generation; specificity.                             |
| **[[Styles vs Settings]]**             | Reinforce the distinction between configuration and appearance.                | Settings = what CAN be changed; Styles = what IS by default; user override capability.                      |
| **[[Root Styles]]**                    | Apply global styles to the body/root element.                                  | `styles.color`, `styles.typography`, `styles.spacing`; equivalent to `body {}` CSS.                         |
| **[[Element Styles]]**                 | Style generic HTML elements globally.                                          | `styles.elements.h1-h6`, `link`, `button`, `caption`, `cite`; targeting semantic HTML.                      |
| **[[Link Styles]]**                    | Define default link appearance.                                                | `styles.elements.link.color`, `typography`, `:hover`, `:focus`; accessibility considerations.               |
| **[[Heading Styles]]**                 | Style heading elements (h1-h6).                                                | `styles.elements.h1` through `h6`; typography scale; color; spacing.                                        |
| **[[Button Styles]]**                  | Define default button appearance.                                              | `styles.elements.button`; color, typography, border, spacing; button block relationship.                    |
| **[[Block-Specific Styles]]**          | Apply default styles to specific blocks.                                       | `styles.blocks.core/paragraph`, `core/heading`, etc.; scoped styling; inheritance.                          |
| **[[Typography in Styles]]**           | Set default typography values.                                                 | `styles.typography.fontSize`, `fontFamily`, `lineHeight`, `fontWeight`; using preset slugs vs values.       |
| **[[Color in Styles]]**                | Set default color values.                                                      | `styles.color.background`, `text`, `link`; using palette slugs vs hex values.                               |
| **[[Spacing in Styles]]**              | Set default spacing values.                                                    | `styles.spacing.padding`, `margin`, `blockGap`; using spacing scale slugs.                                  |
| **[[Border & Shadow Styles]]**         | Define default border and shadow appearance.                                   | `styles.border.color`, `radius`, `style`, `width`; `styles.shadow`; visual depth.                           |
