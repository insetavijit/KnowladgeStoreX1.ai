| **Section**                            | **Interview Learning Goals**                                                   | **Key Technical Details to Study**                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **Spacing Scale**                      | Explain how to create consistent spacing tokens.                               | Spacing presets, padding/margin controls, block gap; creating predictable rhythm.                           |
| **[[Spacing Presets]]**                | Define custom spacing scale values.                                            | `settings.spacing.spacingScale` array; `name`, `slug`, `size` properties; generates `--wp--preset--spacing--*`.|
| **[[Default Spacing Values]]**         | Set default spacing for common use cases.                                      | Small, medium, large presets; typical values (8px, 16px, 24px, 32px, 48px, 64px).                          |
| **[[Fluid Spacing]]**                  | Implement responsive spacing that scales with viewport.                        | Using `clamp()` for spacing values; min/max/preferred spacing; viewport-based scaling.                      |
| **[[Block Gap Control]]**              | Manage spacing between blocks globally.                                        | `settings.spacing.blockGap: true`; default gap value; per-block gap overrides.                              |
| **[[Padding Presets]]**                | Enable padding controls with preset values.                                    | `settings.spacing.padding: true`; using spacing scale for padding; block-level padding.                     |
| **[[Margin Presets]]**                 | Enable margin controls with preset values.                                     | `settings.spacing.margin: true`; using spacing scale for margins; negative margins considerations.          |
| **[[Custom Spacing Size]]**            | Allow custom spacing values beyond presets.                                    | `settings.spacing.customSpacingSize: true`; free-form spacing input; when to allow vs restrict.             |
| **[[Spacing Units]]**                  | Control which units are available for spacing.                                 | `settings.spacing.units` array; px, em, rem, vh, vw, %; unit consistency across theme.                      |
| **[[Vertical Rhythm]]**                | Create consistent vertical spacing patterns.                                   | Baseline grid concepts; consistent spacing multiples; visual rhythm and flow.                               |
| **[[Spacing in Design Systems]]**      | Understand spacing as part of design tokens.                                   | 8-point grid system; spacing scale ratios; consistency across components.                                   |
