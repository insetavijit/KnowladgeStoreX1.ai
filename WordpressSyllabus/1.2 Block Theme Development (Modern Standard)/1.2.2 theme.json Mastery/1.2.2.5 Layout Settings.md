| **Section**                            | **Interview Learning Goals**                                                   | **Key Technical Details to Study**                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **Layout Settings**                    | Explain how to control content width and block alignment.                      | contentSize, wideSize, block gaps, alignment controls; responsive layout behavior.                          |
| **[[Content Size]]**                   | Define the default content width.                                              | `settings.layout.contentSize`; typically 640-800px; affects blocks without alignment.                       |
| **[[Wide Size]]**                      | Define the wide content width.                                                 | `settings.layout.wideSize`; typically 1200-1400px; for align-wide blocks.                                   |
| **[[Allow Editing]]**                  | Control whether users can override layout settings.                            | `settings.layout.allowEditing: true/false`; locking down layout for consistency.                            |
| **[[Block Gap (Spacing)]]**            | Manage vertical spacing between blocks.                                        | `settings.spacing.blockGap: true`; default gap values; responsive gap control.                              |
| **[[Alignment Options]]**              | Understand how alignment relates to layout settings.                           | align-left, align-center, align-right, align-wide, align-full; CSS implementation.                          |
| **[[Custom Units]]**                   | Control which CSS units are available.                                         | `settings.spacing.units` array; px, em, rem, vh, vw, %; restricting unit options.                           |
| **[[Padding & Margin]]**               | Enable padding and margin controls.                                            | `settings.spacing.padding: true`, `margin: true`; block-level spacing controls.                             |
| **[[Responsive Layouts]]**             | Ensure layouts work across devices.                                            | Mobile-first approach; breakpoints; testing on various screen sizes.                                        |
| **[[Container Blocks]]**               | Understand container block layout behavior.                                    | Group block, Cover block; inner block alignment; nested layout contexts.                                    |
