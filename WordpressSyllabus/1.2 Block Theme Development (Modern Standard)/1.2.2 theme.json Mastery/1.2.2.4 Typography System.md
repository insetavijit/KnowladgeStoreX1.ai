| **Section**                            | **Interview Learning Goals**                                                   | **Key Technical Details to Study**                                                                          |
| -------------------------------------- | ------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| **Typography System**                  | Explain how to build a comprehensive type scale and font system.               | Font families, font sizes, fluid typography, line heights, font weights; Google Fonts vs local fonts.       |
| **[[Font Family Presets]]**            | Define custom font families in theme.json.                                     | `settings.typography.fontFamilies` array; `name`, `slug`, `fontFamily` properties; web-safe vs custom fonts.|
| **[[Font Size Scale]]**                | Create a modular font size scale.                                              | `settings.typography.fontSizes` array; `name`, `slug`, `size` properties; responsive sizing strategies.     |
| **[[Fluid Typography]]**               | Implement responsive, viewport-based font sizing.                              | `clamp()` CSS function; `fluid` property in fontSizes; min/max/preferred values; viewport units.            |
| **[[Line Height Control]]**            | Manage line height settings.                                                   | `settings.typography.lineHeight: true`; unitless vs unit-based values; readability optimization.            |
| **[[Font Weight & Style]]**            | Control font weight and style options.                                         | `settings.typography.fontWeight`, `fontStyle`; enabling bold/italic controls; variable fonts support.       |
| **[[Letter Spacing]]**                 | Enable letter spacing (tracking) controls.                                     | `settings.typography.letterSpacing: true`; use cases; accessibility impact.                                  |
| **[[Text Transform]]**                 | Control text transformation options.                                           | `settings.typography.textTransform: true`; uppercase, lowercase, capitalize controls.                        |
| **[[Drop Cap]]**                       | Enable drop cap feature for paragraphs.                                        | `settings.typography.dropCap: true`; styling drop caps; browser compatibility.                               |
| **[[Google Fonts Integration]]**       | Load and use Google Fonts in block themes.                                     | Enqueuing fonts; GDPR considerations; performance optimization; font-display strategies.                     |
| **[[Local Font Files]]**               | Use self-hosted font files.                                                    | `@font-face` declarations; WOFF2 format; font loading performance; licensing considerations.                |
| **[[Typography Presets]]**             | Understand how typography presets generate CSS variables.                      | `--wp--preset--font-family--*`, `--wp--preset--font-size--*`; usage in custom CSS.                          |
