# 1.2.1.3 theme.json Deep Dive

**Overview**
`theme.json` is the control center of a modern WordPress block theme. It consolidates the configuration of features (Settings) and the definition of the design system (Styles) into a single standard file. This replaces dozens of `add_theme_support` calls and custom stylesheet enqueues required in classic themes. By defining the rules here, you ensure the Block Editor (Gutenberg) and the front-end stay perfectly in sync, giving users controls that respect your design constraints.
