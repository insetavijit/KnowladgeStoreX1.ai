# 1.2.1.4 HTML Templates

**Overview**
In block themes, the traditional PHP template hierarchy (`single.php`, `archive.php`) is replaced by **HTML templates** (`single.html`, `archive.html`) located in the `templates/` directory. These files contain **Block Markup**â€”HTML comments that specifically define blocks and their attributes. This shift means logic is handled by blocks (like the Query Loop block) rather than PHP code, enabling users to edit the structure of entire pages directly in the Site Editor.
