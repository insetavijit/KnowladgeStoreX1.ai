**Structure Overview**
The architecture of a Block Theme is fundamentally different from classic WordPress themes. Instead of relying on a hierarchy of PHP files (`header.php`, `footer.php`, `page.php`), Block Themes operate primarily through **HTML templates** and a central configuration file known as **theme.json**. The file structure is simplified, standardizing where templates, parts, and styles reside, which makes the theme more predictable and easier to maintain.
