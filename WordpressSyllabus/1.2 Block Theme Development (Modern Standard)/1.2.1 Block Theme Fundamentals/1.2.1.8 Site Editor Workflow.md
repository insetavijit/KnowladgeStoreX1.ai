# 1.2.1.8 Site Editor Workflow

**Overview**
The **Site Editor** (Appearance > Editor) is the visual interface where users and developers interact with the entire site structure. Unlike the "Content Editor" (for posts/pages), the Site Editor allows modification of Templates, Template Parts, Global Styles, and Navigation menus. For developers, understanding this workflow is critical because changes made here can be exported back to the theme files, acting as a "visual IDE" for theme creation.
