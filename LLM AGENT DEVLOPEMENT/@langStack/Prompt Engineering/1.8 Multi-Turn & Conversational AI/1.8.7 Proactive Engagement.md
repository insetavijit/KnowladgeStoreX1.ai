| **Section** | **Focus** | **Key Utilities / Concepts** | **One-Line Recall** |
| :--- | :--- | :--- | :--- |
| **[[1.8.7.1 Suggestion Generation]]** | Ideas | "You might also like...", related topics | Proactively suggesting unrelated but relevant topics (like YouTube recommendations). |
| **[[1.8.7.2 Anticipating Needs]]** | Prediction | "Since you asked about X, you'll need Y", next step | Predicting the user's next likely problem and solving it before they ask. |
| **[[1.8.7.3 Follow-up Questions]]** | Depth | "Did that work?", "What are you building?" | Asking questions to deepen the engagement or debug the solution. |
| **[[1.8.7.4 Driving the Conversation]]** | Momentum | "Shall we move to the next step?", pacing | Taking charge of the flow prevents the conversation from stalling. |
| **[[1.8.7.5 Checking In]]** | Pulse | "Is this helpful so far?", mid-flight correction | Asking for confirmation during a long process ensures you aren't wasting time. |
| **[[1.8.7.6 Offering Alternatives]]** | Breadth | "Here is one way, but another way is...", choice | Presenting options proactively empowers the user to choose the best path. |
| **[[1.8.7.7 Reminders & Alerts]]** | Timing | "Don't forget to...", scheduled prompts | Setting triggers to re-engage the user after a specific time or event. |
| **[[1.8.7.8 Discovery Prompts]]** | Exploration | "Did you know I can also...", feature education | Teaching the user about capabilities they might not know exist. |
| **[[1.8.7.9 Keeping Engagement High]]** | Retention | Gamification, streaks, curiosity gaps | Using psychological triggers (responsibly) to keep the user interested. |
| **[[1.8.7.10 Balancing Proactivity]]** | Annoyance | "Don't be Clippy", intrusive vs helpful | The fine line between being helpful and being an annoying interruption. |
