# 1.8.8 Conversation Closure

| **Section** | **Focus** | **Key Utilities / Concepts** | **One-Line Recall** |
| :--- | :--- | :--- | :--- |
| **[[1.8.8.1 Summarization]]** | Recap | "To summarize, we decided...", meeting minutes | summarizing the session creates a clean artifact for the user to take away. |
| **[[1.8.8.2 Next Steps]]** | Action | "Here is your todo list", future planning | Ending with clear, actionable items prevents the "what now?" feeling. |
| **[[1.8.8.3 Farewell Protocols]]** | Politeness | "Goodbye", "Have a great day", closing loop | Standard social protocols for closing a connection (even with a bot) feel natural. |
| **[[1.8.8.4 Detecting Exit Intent]]** | Analysis | "Thanks, that's all", "Bye", detecting finish | Recognizing when the user wants to leave so the AI doesn't keep asking "Is there anything else?". |
| **[[1.8.8.5 Feedback Gathering]]** | Improvement | "How did I do?", CSAT score, thumbs up/down | Asking for rating *at the end* is the standard time to collect quality data. |
| **[[1.8.8.6 Saving State]]** | Persistence | "I've saved this to your profile", serialization | Confirming that the work done in the session has been persisted. |
| **[[1.8.8.7 Transitioning]]** | Handoff | "Connecting you to a human", routing | Gracefully handing the conversation off to another agent or human. |
| **[[1.8.8.8 Call to Action]]** | Conversion | "Sign up here", "Buy now", business goal | If the bot has a business purpose, the closure is the time to ask for the sale. |
| **[[1.8.8.9 Re-engagement hooks]]** | Retention | "Check back tomorrow for...", open loops | Planting a seed for the next conversation ensures the user returns. |
| **[[1.8.8.10 Final Polish]]** | Formatting | Clean ending, no trailing tokens, "END" | Ensuring the final output looks finished and professional. |

This table represents a **complete guide to Conversation Closure**, sticking the landing.
